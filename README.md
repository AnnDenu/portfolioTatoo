# Портфолио тату-мастера

Веб-сайт портфолио тату-мастерам VOLOSATIYKENT.

## Концепция

Проект реализует метафору "живого холста", где каждая татуировка получает собственное дыхание в цифровом пространстве. Сайт сочетает минималистичную чистоту фона с интенсивностью и детализацией представленных работ.

## Особенности

- **Кинетическая эстетика**: Плавные анимации, напоминающие течение чернил
- **Тактильный интерфейс**: Микро-анимации и эффекты при взаимодействии
- **Дышащая галерея**: Динамичные элементы портфолио с эффектом "дыхания"
- **Чернильная волна**: Навигация с эффектом расхождения цвета
- **Игольчатая точность**: Четкие контуры и точные переходы
- **Кастомный курсор**: Уникальный курсор с эффектом эхо

## Секции сайта

1. **Hero Section** - Заголовочная секция с анимацией появления
2. **Портфолио** - Интерактивная галерея с фильтрами по стилям
3. **О мастере** - Информация о мастере с статистикой
4. **Процесс работы** - Описание этапов работы
5. **Контакты** - Форма записи и контактная информация

## Анимации и эффекты

- **Breathe Animation**: Эффект дыхания для элементов
- **Fade In Up**: Плавное появление элементов
- **Ink Wave**: Эффект волны в навигации
- **Echo Effect**: Расходящиеся круги при клике
- **Portfolio Filters**: Плавная фильтрация работ
- **Smooth Scroll**: Плавная прокрутка между секциями

## Кастомизация

Все цвета, шрифты и параметры анимаций определены в CSS переменных в начале файла `styles.css`. Вы можете легко изменить:

- Цветовую палитру
- Шрифты
- Скорость анимаций
- Размеры и отступы
